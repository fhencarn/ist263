Usability & Testing Notes

Usability is about how easy and pleasant it is for users to use a website. Key concepts from class include:

  • Learnability – Is it easy to understand the interface the first time?
  • Efficiency – Can users complete tasks quickly once they’ve learned it?
  • Memorability – Can users remember how it works when they come back later?
  • Errors – How many mistakes do users make, and how easy is recovery?
  • Satisfaction – Is the experience pleasant overall?

This page supports usability with clear headings, simple layout, readable text, and descriptive link text.
